# Avito WebSocket Bridge

> Real-time Avito message streaming via Puppeteer + NestJS + WebSocket

A NestJS service that monitors an Avito personal account for new messages using Puppeteer browser automation, and streams them to a frontend client in real time over WebSocket. Includes Cloudflared tunnel support for external access.

## Tech Stack

- **Backend**: NestJS (TypeScript)
- **Browser Automation**: Puppeteer
- **Real-time**: WebSocket (via `@nestjs/websockets`)
- **Tunnel**: Cloudflared (CloudPub)
- **Frontend**: HTML + Vanilla JS (modern design)
- **DevOps**: Docker + docker-compose
